## Getting Started

First, run the development server:

```bash
yarn dev
```

## How to deploy

### Ensure the app can build

Run, the following and fix any errors if there are any

```bash
yarn build
```

### Deploy on Netlify

The app will auto build and deploy when the `main` branch is pushed to.


```bash
git push
```
